while			用于在特定的条件为真的情况下重复执行（先判断后执行）
do-while		用于在特定的条件为真的情况下重复执行，至少执行一次（先执行，后判断）
for			用于在特定的条件为真的情况下重复执行（先判断后执行）

while(表达式)
语句；

do 
{
	语句；
}while(条件)；

for(表达式1；表达式2；表达式3)
{
	语句；
}

break;
可使程序终止循环而执行后面的语句；

continue;
只能用在循环里，作用是跳过循环体中剩余的语句而准备执行下一次循环。
